O que é o node.js ?:

- npm é um gerenciador de pacotes

Variáveis e Constantes: 
- variável é um lugar de memória
- As variáveis não tem um tipo definido
- camelCase

- Uma constante é um valor atribuído a uma variável que não pode ser alterado(const)

Criação de uma variável(let e var)
- o let é uma variável local e mais recomendada
- a var é uma variável global e é uma versão mais antiga

Console.log:
- saída de dados
- console.log(nomeVariavel)

%d = referenciar um inteiro
`${nomeVariavel}` = adicionar uma variável no meio do texto(concatenar)

Tipos de variável:

- String: texto
- Number: float + number 
- Boolean: verdadeiro(1) ou falso(0) 
- BigInt: Números inteiros grandes
- Undefined: indefinido 
- Null: valor vazio
- Symbol: instâncias são únicas e imutáveis
- Object: coleta de dados com chaves-valor {}

- typeof(): verificar o tipo da variável
